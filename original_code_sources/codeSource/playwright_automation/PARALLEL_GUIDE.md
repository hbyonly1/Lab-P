# Playwright 并行自动化使用指南

## 新功能：多窗口并行处理

现在支持多个学号同时在独立的浏览器窗口中处理，每个浏览器实例完全独立。

## 配置文件

编辑 `config.json`:

```json
{
  "student_ids": [
    "2021001",
    "2021002", 
    "2021003"
  ],
  "max_parallel_browsers": 3,
  "target_url": "http://10.25.77.60:8001/ReportStudent/CompleteReport/"
}
```

### 配置说明

- `student_ids`: 要处理的学号列表
- `max_parallel_browsers`: 最大并行浏览器数量（建议不超过5）
- 其他配置保持不变

## 运行方式

### 方式1: 并行处理（推荐）

```bash
python parallel_automation.py
```

**特点**:
- 多个浏览器窗口同时运行
- 每个浏览器完全独立（独立cookie、session、localStorage）
- 可控制最大并发数
- 速度最快

### 方式2: 顺序处理

```bash
python advanced_automation.py
```

**特点**:
- 一个接一个处理
- 适合调试
- 速度较慢

## 独立浏览器实例

每个学号使用独立的浏览器配置文件：

```
browser_profiles/
├── 2021001/          # 学号1的浏览器数据
│   ├── cookies/
│   ├── localStorage/
│   └── ...
├── 2021002/          # 学号2的浏览器数据
└── 2021003/          # 学号3的浏览器数据
```

**优势**:
- Cookie完全隔离
- Session独立
- 不会互相干扰
- 可以同时登录多个账号

## 并发控制

通过 `max_parallel_browsers` 控制同时运行的浏览器数量：

```json
"max_parallel_browsers": 3
```

**建议值**:
- 电脑配置较低: 1-2
- 普通配置: 3-5
- 高配置: 5-10

## 执行流程

```
读取配置文件
    ↓
获取学号列表 [2021001, 2021002, 2021003]
    ↓
创建并行任务（最多3个同时运行）
    ↓
┌─────────────┬─────────────┬─────────────┐
│ 浏览器1     │ 浏览器2     │ 浏览器3     │
│ 学号2021001 │ 学号2021002 │ 学号2021003 │
│ 独立profile │ 独立profile │ 独立profile │
└─────────────┴─────────────┴─────────────┘
    ↓           ↓           ↓
  自动化      自动化      自动化
    ↓           ↓           ↓
  截图        截图        截图
    ↓
生成统一报告
```

## 性能对比

| 学号数量 | 顺序处理 | 并行处理(3并发) | 提升 |
|---------|---------|----------------|------|
| 3个 | 6分钟 | 2分钟 | 3倍 |
| 6个 | 12分钟 | 4分钟 | 3倍 |
| 9个 | 18分钟 | 6分钟 | 3倍 |

## 注意事项

1. **资源占用**: 每个浏览器占用约500MB内存
2. **网络带宽**: 并行请求会增加网络负载
3. **服务器压力**: 避免并发数过高导致服务器限流
4. **浏览器窗口**: 非无头模式会显示多个窗口

## 故障排除

### 问题1: 内存不足

**解决**: 减少 `max_parallel_browsers` 值

```json
"max_parallel_browsers": 2
```

### 问题2: 浏览器窗口太多

**解决**: 使用无头模式

```json
"headless": true
```

### 问题3: 某个学号失败

**解决**: 查看报告中的错误详情，单独处理失败的学号

## 清理浏览器数据

如需重新开始，删除浏览器配置文件：

```bash
# Windows
rmdir /s /q browser_profiles

# Linux/Mac
rm -rf browser_profiles
```

## 完整示例

`config.json`:
```json
{
  "userscript_path": "../build/bundle.user.js",
  "target_url": "http://10.25.77.60:8001/ReportStudent/CompleteReport/",
  "student_ids": [
    "2021001",
    "2021002",
    "2021003",
    "2021004",
    "2021005"
  ],
  "max_parallel_browsers": 3,
  "headless": false,
  "timeout": {
    "automation_complete": 120000
  }
}
```

运行:
```bash
python parallel_automation.py
```
